# Course Overview

## Learning Scenario

This course prepares you to design and implement real-world IoT solutions by combining hands-on physical computing with cloud-based data services.

### The Smart Home Challenge

Throughout this course, you'll progressively build a smart home monitoring system that:

- Collects environmental data (temperature, motion, air quality)
- Processes sensor data using embedded systems
- Communicates with cloud services
- Provides real-time monitoring and alerts
- Stores and analyzes historical data

## 18-Week Course Structure

### Phase 1: Physical Computing Foundations (Weeks 1-8)

**Week 1: Electronics Basics**
- Voltage, current, and resistance
- Reading circuit diagrams
- Using a multimeter

**Week 2: Breadboards & Buttons**
- Breadboard wiring
- Digital inputs with pull-up/pull-down resistors
- ESP32 GPIO basics

**Week 3: Temperature Sensing**
- DHT11/DHT22 sensors
- I2C and serial protocols
- Data validation

**Week 4: Motion, Gas & Serial Communication**
- PIR motion sensors
- MQ-series gas sensors
- Serial debugging and monitoring

**Week 5: Actuators & Servos**
- Controlling motors and servos
- PWM signals
- Power management

**Week 6: Signal Processing**
- Filtering noisy sensor data
- Moving averages
- Threshold detection

**Week 7: FreeRTOS on ESP32**
- Real-time operating systems
- Task scheduling
- Inter-task communication

**Week 8: IoT House Synthesis**
- Integrating multiple sensors
- Complete system design
- Preparation for cloud connectivity

### Phase 2: Cloud Foundations (Weeks 9-12)

**Week 9-10: AWS Cloud Practitioner Essentials**
- Cloud computing concepts
- AWS services overview
- Security and compliance

**Week 11: IoT Core & Device Management**
- MQTT protocol
- Device shadows
- Fleet management

**Week 12: Data Storage & Analytics**
- DynamoDB basics
- S3 storage
- Basic data analytics

### Phase 3: Integration & Projects (Weeks 13-18)

**Week 13-14: Full Stack IoT**
- ESP32 to AWS integration
- Real-time data pipelines
- Dashboard creation

**Week 15-16: Advanced Features**
- Machine learning integration
- Predictive maintenance
- Automated alerts

**Week 17-18: Final Project**
- Custom IoT solution design
- Implementation and testing
- Presentation and documentation

## Assessment Structure

1. **Weekly Labs** (40%) - Hands-on practical work
2. **Mid-term Project** (20%) - Week 8 synthesis project
3. **AWS Certifications** (15%) - Cloud foundations knowledge
4. **Final Project** (20%) - Complete IoT system
5. **Participation & Reflection** (5%) - Engagement and learning journal

## Learning Outcomes

By the end of this course, you will be able to:

- Design and implement embedded systems using ESP32
- Interface with various sensors and actuators
- Develop firmware using FreeRTOS
- Connect devices to AWS IoT Core
- Store and analyze IoT data in the cloud
- Build complete end-to-end IoT solutions
- Apply security best practices for IoT systems
