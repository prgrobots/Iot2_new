# SimuLearn Labs List

## Available SimuLearn Laboratory Exercises

This page lists all available SimuLearn labs with search strings for easy access.

### Physical Computing Labs (Weeks 1-8)

#### Week 1: Electronics Basics
- **Lab:** "Basic Electronics Circuits"
- **Search:** `SimuLearn electronics fundamentals`
- **Topics:** Voltage, current, resistance, Ohm's Law

#### Week 2: Breadboards & Digital I/O
- **Lab:** "Breadboard Wiring and Button Inputs"
- **Search:** `SimuLearn breadboard button ESP32`
- **Topics:** Breadboard connections, pull-up/down resistors, GPIO

#### Week 3: Temperature Sensors
- **Lab:** "DHT Sensor Integration"
- **Search:** `SimuLearn DHT temperature humidity sensor`
- **Topics:** Digital sensors, I2C, data reading

#### Week 4: Motion & Gas Sensors
- **Lab:** "PIR and MQ Sensors"
- **Search:** `SimuLearn PIR motion MQ gas sensor`
- **Topics:** Motion detection, analog sensors, ADC

#### Week 5: Actuators & Servos
- **Lab:** "Servo Motor Control"
- **Search:** `SimuLearn servo motor PWM control`
- **Topics:** PWM signals, servo control, power management

#### Week 6: Signal Processing
- **Lab:** "Sensor Data Filtering"
- **Search:** `SimuLearn signal processing filter noise`
- **Topics:** Moving averages, noise reduction, thresholds

#### Week 7: FreeRTOS
- **Lab:** "Real-Time Task Management"
- **Search:** `SimuLearn FreeRTOS ESP32 tasks`
- **Topics:** Task creation, scheduling, synchronization

#### Week 8: Integration Project
- **Lab:** "Multi-Sensor IoT System"
- **Search:** `SimuLearn IoT integration multi-sensor`
- **Topics:** System design, integration, testing

### Cloud & AWS Labs (Weeks 9-12)

### Advanced Integration Labs (Weeks 13-18)

#### Week 13: ESP32 to AWS

#### Week 14: Real-Time Dashboards

#### Week 15: Machine Learning

## How to Access SimuLearn Labs

1. Log in to SimuLearn platform (credentials provided by instructor)
2. Use the search strings provided above
3. Complete pre-lab readings and quizzes
4. Follow step-by-step lab instructions
5. Submit lab reports through the course portal

## Lab Completion Requirements

- Complete all required steps
- Answer quiz questions
- Submit screenshots or code
- Write brief reflection (50-100 words)

## Technical Support

- SimuLearn Help Desk: Available 24/7

- Peer support: Course discussion forum
