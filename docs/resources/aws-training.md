## AWS IoT Learning Plan: Foundations Overview

**Official Course:** [AWS IoT Learning Plan: Foundations](https://skillbuilder.aws/learning-plan/PD7V83RNUR/aws-iot-learning-plan-foundations/2XH6DKH3PF)

**Total Duration:** 7 hours 50 minutes across 5 courses

**Courses:**

1. **Getting Started with AWS IoT** – Introduction to AWS IoT, use cases, and benefits; key concepts and core services for industrial applications

2. **Securely Connecting AWS IoT Devices to the Cloud** – Device-to-cloud authentication using X.509 certificates; secure MQTT communication and IoT policy configuration

3. **Handling AWS IoT Device Data and States** – MQTT protocol, publish-subscribe model, IoT Rules Engine for processing telemetry, and Device Shadows for offline state management

4. **Managing AWS IoT Devices at Scale** – Large-scale IoT architecture, fleet management, device indexing, and OTA updates for hundreds or thousands of devices

5. **Analyzing, Visualizing, and Gaining Insights from IoT Devices** – Real-time analytics with QuickSight, Timestream, Grafana, and IoT Analytics for monitoring and insights

## Additional AWS Courses (Weeks 12–17)

**OTA Updates in Edge Devices** (NMTAFE Sandbox, 1 hour) – [Link](https://skillbuilder.aws/learn/87ZT9986F5/aws-simulearn-ota-updates-in-edge-devices/3FJ8CA6NBD). AWS SimuLearn scenario: manage remote updates for IoT applications across industrial edge devices without manual intervention. Learn to deploy code changes, sensor updates, and firmware patches securely to hundreds of haul trucks in the field using AWS device management services.

**ETL at the Edge** (NMTAFE Sandbox, 1 hour) – [Link](https://skillbuilder.aws/learn/4KEZK2TQMN/aws-simulearn-etl-at-the-edge/JHZ9HGNXDG). AWS SimuLearn scenario-based lab: solve real-world vehicle data collection challenges in areas with poor internet connectivity. Build ETL (extract, transform, load) solutions at the edge to manage data locally, ensure continuity, and synchronize with cloud when connectivity returns; directly applicable to haul trucks in remote mining areas.

**Lab – Deploying AWS IoT Greengrass Components** (NMTAFE Sandbox, 1 hour) – [Link](https://skillbuilder.aws/learn/XPF9AXGEFP/lab--deploying-aws-iot-greengrass-components/3HX169SQDK). Hands-on lab setting up a Greengrass core device, creating and deploying custom components to run edge logic locally. Learn to use Greengrass CLI and AWS IoT console to manage deployments; enables offline-first operation and real-time processing on ESP32 devices.

**Connecting Systems and Machines to AWS for Industrial Manufacturing** (AWS Free Course, 45 min) – [Link](https://skillbuilder.aws/learn/WF2SG8HBXC/connecting-systems-and-machines-to-aws-for-industrial-manufacturing/DC6TKMVC1S). Learn how to bring machine data into the cloud as a prerequisite for advanced data innovation. Topics include data lakes, data collection and normalization, industrial data security, and real-world manufacturing use cases. Uses AWS IoT Core, SiteWise, Greengrass, and other OT/IT integration services.

## How to Access AWS Skill Builder (Enrollment Process)

**Step 1: Instructor Adds You to Sandbox**
Your instructor will add you to the NMTAFE AWS Academy Sandbox. You'll receive an email from Canvas LMS (not AWS) with activation instructions.

**Step 2: Activate AWS Skill Builder Subscription via Canvas**
1. Log in to Canvas and open any AWS Academy course
2. Select "Modules" menu
3. Scroll to the last module "AWS T&C Resources"
4. Review FAQs and click "Activate Your AWS Skill Builder Offer"
5. Take the survey and consent to information sharing
6. Click "Accept" to join the AWS Academy Skill Builder Program
7. **Note:** Your email must be verified in Canvas to be eligible

**Step 3: Wait for Activation (up to 3 days)**
After submission, AWS will process your request. You'll receive an email from AWS Skill Builder with login instructions.

**Step 4: Access AWS Skill Builder**
- Login using the email from AWS Skill Builder
- You'll automatically have access to the AWS IoT Learning Plan: Foundations courses
- You may also see additional AWS Cloud Quest courses (Cloud Practitioner, Serverless Developer, Solutions Architect)
