# Welcome to the IoT Course

## Physical Computing to Cloud

This comprehensive 18-week course takes you on a journey from basic electronics to cloud-connected IoT systems.

### Course Structure

- **Weeks 1-8**: Physical Computing with ESP32
- **Weeks 9-12**: Cloud Services and AWS Foundations
- **Weeks 13-18**: Advanced IoT Integration and Projects

### Getting Started

Navigate through the course content using the menu above. Each week builds upon previous concepts, so we recommend following the sequence.

## Prerequisites

- Basic programming knowledge
- Curiosity about electronics and IoT systems
- ESP32 development kit (provided)
