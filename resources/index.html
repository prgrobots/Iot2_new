    Wokwi Simulator¶

    URL: wokwi.com

    Free Arduino/ESP32 simulator used in Weeks 2-4 before transitioning to real hardware.

    Features: - ESP32, Arduino Uno, Pi Pico boards - Extensive sensor library - Serial monitor - Share projects via URL

    Hardware Kit¶

    Keyestudio ESP32 IoT Starter Kit (provided Week 5)

    Includes: - ESP32-WROOM-32 development board (2×) - Breadboards - DHT11 temp/humidity sensor - MQ-2 gas sensor - Flame sensor - PIR motion sensor - RFID-RC522 module + cards - GY-521 (MPU6050) accelerometer - Rain sensor - Photoresistor - HC-SR04 ultrasonic sensor - SG90 servo motor - OLED display (SSD1306) - RGB LEDs, resistors, buttons, jumper wires

    Not Included (purchase separately or improvise): - 20×4 LCD I²C display (for pit station) - Second ESP32 (for pit station) – may be provided Week 16 - Piezo buzzer

    Software Tools¶

    Arduino IDE¶

    Download: arduino.cc/software

    Required Libraries: - DHT sensor library (Adafruit) - MFRC522 (RFID) - MPU6050 (Electronic Cats) - Adafruit SSD1306 - Adafruit GFX - PubSubClient (MQTT) - ArduinoJson - RTClib - ESP32Servo - LiquidCrystal_I2C

    Install via: Tools → Manage Libraries

    For advanced users: platformio.org

    AWS CLI¶
