    AWS Foundations Resources¶

    AWS Cloud Practitioner Essentials¶

    This page provides direct links to the AWS training courses that form the cloud computing foundation of this IoT course.

    Required AWS Training Courses¶

    1. AWS Cloud Practitioner Essentials¶

    Duration: ~6 hours
    Description: Introduction to AWS Cloud concepts, services, security, architecture, pricing, and support.

    Access: AWS Skill Builder - Cloud Practitioner Essentials

    2. AWS Cloud Quest: Cloud Practitioner¶

    Duration: ~12 hours
    Description: Interactive, game-based learning for practical AWS skills.

    Access: AWS Cloud Quest

    3. AWS IoT Core Foundations¶

    Duration: ~3 hours
    Description: Introduction to AWS IoT Core services and device connectivity.

    Access: AWS Skill Builder - Search for "IoT Core"

    4. Getting Started with AWS IoT Analytics¶

    Duration: ~1 hour
    Description: Learn to process and analyze IoT data.

    Access: AWS Skill Builder - IoT Analytics

    5. AWS Security Fundamentals¶

    Duration: ~3 hours
    Description: AWS security concepts and best practices for IoT.

    Access: AWS Skill Builder - Security Fundamentals

    Certification Path¶

    AWS Certified Cloud Practitioner¶

    • Exam Code: CLF-C02
    • Cost: $100 USD
    • Duration: 90 minutes
    • Format: Multiple choice and multiple response

    Study Resources: - AWS Skill Builder training courses (above) - AWS Certification exam guide - Practice exams
